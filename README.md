#### Controls  
Move - WASD, Left Stick  
Look - Mouse, Right Stick  
Jump - Space  
Run - Shift, Left Stick Click  
Toggle noclip - G, Triangle  
Go to start - Home, Circle  
Start/stop music - T, Square  
Save - [, L1  
Load - ], R1  
Throw ball - Left Click, X  
Cycle shaders - F, Right Stick Click
